# Kodeboard

> AI-powered project management for developer teams.

## Quick Start

```bash
npm install
npm run docker:up
npm run db:migrate
npm run dev
```

## Stack

- **Frontend:** React 18, TailwindCSS, Zustand, Framer Motion
- **Backend:** Express, Prisma, Socket.IO
- **Database:** PostgreSQL, Redis
- **AI:** OpenAI, Anthropic
